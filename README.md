# SAPLE: Swift Analysis Pipeline for Lightcurve Extraction
Welcome to SAPLE, the Swift Analysis Pipeline for Lightcurve Extraction. The codes contained in this folder will allow you to: 

1. Run the UVOT analysis pipeline on any number of observations you have for one astrophysical source & extract the absorption corrected specific fluxes;

2. Run the XRT analysis pipeline on any number of observations you have for one astrophysical source & extract the powerlaw spectral information (flux and photon index);
